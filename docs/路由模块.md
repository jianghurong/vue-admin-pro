1. 路由

```js
// @todo layout与router的原理解释
路由包裹(除登录路由与404路由其余都为主页的子路由) 
routes = [
  {
    path: '/',
    name: 'home',
    component: () => ('@/layouts/BasicLayout.vue'),
    children: [
      {
        path: '/dash',
        name: 'dash',
        component: () => import('@/views/Dash')
      }
    ],
  },
  {
    path: '/login',
    name: 'login',
    component: () => import('@/views/Login')
  }
]
```
